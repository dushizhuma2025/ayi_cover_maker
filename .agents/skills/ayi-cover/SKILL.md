# 阿一封面制作 Skill (ayi-cover)

`ayi-cover` 是本项目专属的封面提示词生成技能。能够基于输入的文章或草稿，自动萃取核心意图与视觉隐喻，匹配最契合的视觉风格，并一次性产出多画幅比例统一的中文 AI 绘图提示词 markdown 文档。

---

## 核心工作流程 (Workflow)

### 1. 主标题与副标题确定
- **规则 A（用户显式指定）**：若用户指令中明确提供了 `主标题：xxx` 或 `副标题：yyy`，**必须严格无条件使用用户指定的标题与副标题文本**，不得擅自修改、增删字词。
- **规则 B（自动提炼抽取）**：若用户未指定主标题或副标题，则从输入文章内容中自动提取：
  - **A层主标题**：高冲击力短标题（极简、抓眼）。
  - **B层完整标题/副标题**：完整含义补充说明。

### 2. 内容萃取与视觉隐喻分析
- 从文章中提炼一个直接、易懂的核心场景或符号隐喻（例如修车说明书、交接棒、咨询数据图表等）。
- 确定视觉色调与情绪基调（如高效、专业、热烈、沉静等）。

### 3. 设计风格（Style）决策
- **规则 A（用户显式指定）**：若用户指令中指定了某中风格（如指定“蓝黄波点波普风”或 `blue-yellow-pop-dot`），直接使用该风格。
- **规则 B（指定平台匹配）**：若用户指定了平台（如“公众号”），从风格库中挑选最适配该平台的风格（如公众号优先推荐 `business-magazine-front-page` 或 `black-red-silhouette`）。
- **规则 C（自动智能挑选）**：若用户未指定风格与平台，阅读 `references/style-catalog.md` 索引，从 31 种内置风格中根据文章主题（科技/商业/科研/艺术/教程等）自动挑选最契合的 1 种。

### 4. 画幅尺寸与平台覆盖决策
- **规则 A（用户指定尺寸/平台）**：若用户指定了特定比例（如 `3:4、4:3、1:1、2.35:1`）或特定平台，**仅生成用户指定的比例/平台提示词**。
- **规则 B（默认全平台覆盖）**：若用户没有指定图片比例或平台，默认覆盖抖音、公众号、小红书、B 站全平台，**同时生成 3:4、4:3、2.35:1、1:1、16:9 全部 5 种尺寸**。
- **统一延展原则**：同一批产出的所有尺寸提示词，必须使用**相同的视觉风格、配色方案、视觉隐喻与主副标题文案**，仅针对画幅比例与构图版式进行重排。

### 5. 中文提示词编译
- 提示词语言：**全中文自然语言描述**（针对 ChatGPT / GPT-4o / Gemini / Nano / 即梦等大模型优化）。
- 依据 `references/prompt-blueprint.md` 的规范编译提示词。
- 提示词中**绝对不得出现“竖版”、“横版”、“极扁横版”等代表图片画幅比例的形容字样**。
- 画幅比例说明**必须单独独占在提示词最后一行**（如 `画幅比例为3:4`），便于使用者手动切换 UI 比例按钮时一键擦除。

### 6. 输出存盘规则
- **路径决策**：
  - 若用户显式指定了输出路径，保存至指定路径。
  - 若用户未指定输出路径，自动保存在**与输入内容文件相同的相对目录下**。
- **文件名规范**：`{输入文件相同文件名}_C.md`。例如输入文件为 `E:\Document\notes\article.md`，则产出文件为 `E:\Document\notes\article_C.md`。
- **MD 文件内结构**：
  - 最顶部：写入 `# 封面设计方案说明`，注明采用的设计风格名称、Style ID 及选定理由。
  - 正文：每个尺寸使用二级标题（如 `## 尺寸：3:4 (适用于：小红书/抖音封面)`）进行分隔。
  - 代码框：将具体的提示词用 Markdown 代码框（```text ... ```）包裹。

## 示例 MD 输出结构

```markdown
# 封面设计方案说明

- **选用设计风格**：咨询报告视觉 (`consulting-report-visual`)
- **选用理由**：文章拆解 300 个爆款选题与流量背后的情绪需求匹配，咨询报告风格极具专业数据拆解质感。
- **核心视觉隐喻**：深藏青背景下的咨询报告数据分析视图，高亮展示痛点匹配节点。

---

## 尺寸：3:4 (适用于：小红书/抖音封面)

```text
请生成一张小红书封面图。采用咨询报告视觉风格。
【背景色调与材质】：深藏青色极简商务背景，带有细腻的几何网格线。
【文字排版】：画面上方为纯白粗体中文大标题“七月爆款文章分析报告”，下方搭配亮橙色字幕“流量背后的唯一逻辑：把读者最焦虑的事说到位”。
【视觉主体】：画面正中央为一个高亮度的咨询报告数据统计面板，包含三个关键数据柱状图与人群痛点匹配节点。
【画面细节与质感】：麦肯锡咨询报告图例质感、矢量数据图表、橙色高亮数据节点。
【禁止元素】：禁止出现杂乱插画、3D卡通形象、做旧旧报纸颗粒及夸张表情包。
整体画面呈现出严谨、高级的商业数据拆解质感。
画幅比例为3:4
```
