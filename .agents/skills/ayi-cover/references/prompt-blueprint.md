## 2. 生成规则与约束

1. **一致性原则**：
   - 风格描述、色彩系统、核心隐喻主体、主副标题文案在所有尺寸的 Prompt 中必须**保持 100% 相同**。
   - 仅针对不同比例调整排版位置（如 3:4 上下排布、2.35:1 左右排布、16:9 避开右下角）。
2. **严禁包含“竖版/横版”等比例形容词**：
   - 开头及正文中**严禁出现“竖版”、“横版”、“极扁横版”等任何代表图片画幅比例的形容词**。
3. **后置画幅比例**：
   - 每个代码框内的提示词文本，**最后一行独占写明画幅比例**，格式统一为：`画幅比例为X:Y`（方便用户若使用外部 UI 界面手动设置比例按钮时一键擦除）。
4. **中文优化**：
   - 主标题文案明确用引号包覆，提示 AI 保持中文精准渲染。
